# Lemon PDF Studio — 测试用例

> 基于当前实现的全部功能模块（PasswordHub / MultiFileMerger / PageSplitter / EditorStudio / 通用能力）
> 编写时使用术语统一说明：
> - **"源文件"**：用户加载进任一模块的 PDF / 图片文件
> - **"当前页"**：中央预览面板正在显示的页
> - **"活动页"**：未标记 `deleted` 的页（参与最终导出）

---

## 0. 测试准备

| 项目 | 说明 |
| --- | --- |
| 测试 PDF A | 10 页标准 A4 竖版纯文本 PDF（约 1 MB） |
| 测试 PDF B | 50 页混排 PDF（含 0° / 90° / 180° / 270° 混合旋转） |
| 测试 PDF C | 加密 PDF — 仅 Owner Password（限制复制/打印） |
| 测试 PDF D | 加密 PDF — User Password（需密码才能打开） |
| 测试 PDF E | 损坏 / 解析告警 PDF（存在无效对象，但能正常打开） |
| 测试图片 | JPG / PNG / WebP 各 1 张 |
| 浏览器 | Chrome 最新版，DevTools 打开 Console |
